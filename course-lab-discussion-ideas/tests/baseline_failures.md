# Course Lab Discussion Ideas Baseline Failures

Date: `2026-04-04`

These RED-phase notes capture the failures the new standalone skill package must prevent.

- Broad browsing happens every run because experiment-local permanent memory is ignored.
- Retained candidates skip targeted enrichment even though the approved contract requires one or two targeted rounds.
- Reference reports are treated as optional ambient context instead of required parent-passed inputs.
- The builder drifts into synthesis-owned prose instead of staying artifact-only and idea-oriented.
- Routine comparison, uncertainty, anomaly, and completeness material is wrongly promoted into discussion ideas even when it is just normal report work.
- Search is triggered for routine report obligations instead of only for novelty-qualified, beyond-handout extra-analysis directions.
- A run tries to manufacture a minimum number of ideas instead of allowing zero valid non-routine ideas.
- Repeated ideas overwrite or erase provenance instead of being deduplicated while preserving support notes and reuse status.
