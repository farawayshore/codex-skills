# Course Lab Discussion Synthesis Baseline Failures

- No standalone `course-lab-discussion-synthesis` package exists yet under `/root/.codex/skills/`.
- There is no local builder that validates approved synthesis inputs before writing discussion artifacts.
- There is no enforcement that parent-passed `reference-report` inputs are required.
- There is no first accepted-ideas run marker for bounded web enrichment.
- There is no rerun rule that suppresses repeated broad browsing and switches to targeted gap filling only when ideas are still weak.
- There is no structured `discussion_blocks` plus `support_records` output contract for `course-lab-final-staging`.
- There is no explicit protection against mutating `main.tex` directly from the synthesis layer.
- There is no local prompt that keeps synthesis aligned to approved directions instead of inventing new major discussion themes.
